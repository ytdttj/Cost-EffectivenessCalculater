# 性价比计算器

 抠搜的人都喜欢比较哪个商品比较划算（指我自己和我女朋友）

## 功能

- 输入两件商品的重量/体积和价格，计算每件商品的单位价格（即每克或每毫升的价格）。比较两件商品的单位价格，并显示哪件商品的性价比更高。

## 使用指南

1. 运行应用，在主界面输入两件商品的重量/体积和价格。

2. 点击“计算”按钮，应用将计算并显示哪件商品的性价比更高。
